На вход поступает одна из бинарных операций:

ADD X Y
SUB X Y
MUL X Y
DIV X Y
Необходимо вывести результат применения операции. В случае деления на 0 выводить 0.
Доп требования:
1. Каждая операция должна быть реализована в отдельной функции, принимающей аргументы и возвращающей результат
2. Поиск операции должен осуществляться по словарю (наименование операции, указатель на функцию операции)

Input format
команда и операнды

Output format
результат операции
