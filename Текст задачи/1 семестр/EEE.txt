Задача E

Напишите программу, которая проверяет является ли число в десятичной записи палиндромом.
Проверка на палиндром должна происходить в отдельной функции.
Проверка на палиндром должна происходить путем разложения числа на цифры

Input format
На входе натуральное число

Output format
На выходе строка true, если палиндром, иначе - false

Examples
Input
33

Output
true
