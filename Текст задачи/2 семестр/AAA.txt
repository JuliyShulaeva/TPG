Задача A
Есть 3 реализации декодера: base64, plain text, zip + base64

На вход программе подаются строки. Каждая строка состоит из префикса(id декодера) и данных. Декодируйте строки и выведите на поток стандартного вывода. Каждый декодер - отдельный класс, реализующий интерфейс декодирования. Соответствующие id декодеров:

1 - plain text

2 - base64

3 - zip + base64

Примеры
Входные данные
1hello
2aGVsbG8K
Результат работы
hello
hello
